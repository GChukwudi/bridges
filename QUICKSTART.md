# Quick Start Guide

## Get Started in 3 Steps

### 1. Install Dependencies
Open your terminal in the project folder and run:
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Open in Browser
The app will be running at: `http://localhost:5173`

## What You'll See

- **Homepage** - Hero slideshow with testimonials
- **SME Directory** - Filterable business listings
- **Opportunities** - Funding, training, and partnerships
- **Resources** - Social media guides and tips
- **Trade Fair** - Event information and gallery
- **Dashboard** - Profile management system

## Changing Languages

Click the language buttons (EN/RW/FR) in the top navigation bar.

## Navigation

The left sidebar (in the instructions) is actually the main top navigation with these links:
- Home
- Join as an SME (SME Directory)
- About Us
- Trade Fair
- Gallery
- Opportunities
- Resources
- Dashboard

## Customization Tips

1. **Colors**: Edit CSS variables in `src/index.css` (`:root` section)
2. **Content**: Update arrays in respective page files
3. **Images**: Replace Unsplash URLs with your own images
4. **Translations**: Add/edit in `src/translations.js`
